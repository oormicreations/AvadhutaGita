Help

This app reads the Ashtavakra Gita. Touch the Sage to start.

It will read the verses at a set schedule.
Use Settings to set the schedule, notification sound and speech. You can set it to read sequentially or randomly.
Try the auto read mode to read it without a schedule or just swipe to get next/previous verse.
Click the Go-To button (curved arrow) to go to a particular verse.

This app is open source and free. Please share :)
