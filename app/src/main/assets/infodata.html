Help

This app is a reader for Avadhuta Gita. It is the famous nondualism text composed by Dattatreya.

You can read it by pressing the next and previous buttons or can set it on auto mode or on a schedule. Use settings to a set schedule. Touch the Sage to start. Or use the arrow buttons to flip pages.
Use Settings to set the notifications, sounds and speech. You can set it to read sequentially or randomly.

Click the Go-To Verse in menu to go to a particular verse.

It can also read it aloud. Use the Mute button on top to mute it, or mute it from Settings menu.

Verses in both English and Sanskrit can be read. It needs Hindi TTS data to be installed for Google TTS. You can set that in Android settings for Language and input.

This app is open source and free. Please share :)
